Overview

This executable test suite is based on the following OGC specifications:

  • OGC® Web Coverage Service 2.0 Interface Standard -Earth Observation Application Profile
  • Compliance Test Language (CTL) 0.5

What is tested

  • GetCapabilities (KVP)
  • DescribeCoverage (KVP)
  • DescribeEOCoverageSet (KVP)

Further extensions are only tested when the GetCapabilities response indicates that the extensions are supported.

The user is prompted for the end point URL of the server. If there is no server at that location, the testing stops.
